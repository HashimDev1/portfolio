# 🌟 Premium Personal Portfolio - Muhammad Hashim

A modern, highly responsive, and visually stunning personal portfolio built with React, Vite, and Tailwind CSS. Featuring glassmorphism design, custom micro-animations with Framer Motion, and mobile-friendly optimization.

---

## ✨ Features

- **📱 Fully Responsive Design**: Seamlessly adapted for desktops, tablets, and mobile devices (with optimized touch elements and automatic navigation layout adjustments).
- **🎨 Modern Glassmorphism UI**: Curated aesthetic with custom gradients, warm tones, and polished typography.
- **⚡ Supercharged Performance**: Powered by Vite for near-instant hot module replacement (HMR) and fast loading times.
- **✨ Micro-Animations**: Smooth scroll-reveal transitions, custom hover highlights, and fluid floating motion designs powered by Framer Motion.
- **📂 Structured Project Architecture**: Clean, reusable component-based React structure.

---

## 🛠️ Tech Stack

- **Frontend**: React 19, JavaScript (ES6+), HTML5, CSS3
- **Styling**: Tailwind CSS (v4)
- **Animations**: Framer Motion
- **Icons**: Lucide React & React Icons
- **Build Tool**: Vite

## 🚀 Getting Started

Follow these steps to run the portfolio locally on your machine and access it on your mobile device.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HashimDev1/premium-portfolio.git
   cd premium-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

### 📱 View on Mobile (Localhost Sharing)

The project is configured to automatically expose the server to your local network. 

1. Ensure both your computer and your phone are connected to the **same Wi-Fi network**.
2. Run `npm run dev`.
3. In your terminal, look for the **Network** address:
   ```text
   ➜  Network: http://<your-local-ip>:<port>/
   ```
4. Open that URL in your mobile browser.
